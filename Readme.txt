S.O.P

Step1：Scan your QRCODE 

Step2 - 1：If QRCODE is wrong , will notice you “ The QRCODE is wrong ” , you should scan again.

Step2 - 2：If QRCODE is right ,will enter main window

Step3 ：When you enter the main window , you will see the “Open Cap ” Button or “Logout” Button

Step4 -1 ：If you press the “Open Cap ” Button ,trash can cap will open 

Step4-2 ： If you press the “Logout ” Button ,you will return Login Window , you can scan other QRCODE.


Official website：https://loveangelritah.wixsite.com/smarttrashcan


Program explanation：

Rasp_Ctr.py : Raspberry Pi control motor and UDP communication

qzxing-master : QRCODE decode library

Qt : is Qt interface main code
